# 연습문제 2.1
다음 중 어느 점이 곡선 y<sup>2</sup> = x<sup>3</sup> +5x + 7 위에 있는가?
<pre>
    (2, 4), (-1, -1), (18, 77), (5, 7)
</pre>

```
>>> def on_curve(x, y):
...     return y**2 == x**3 + 5*x + 7
>>> print(on_curve(2, 5))
False
>>> print(on_curve(-1,-1))
True
>>> print(on_curve(18,77))
True
>>> print(on_curve(5,7))
False
```
